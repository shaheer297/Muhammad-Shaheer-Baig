### 🔌 Smart Universal Charger

Control logic for a smart battery charger with **CV (Constant Voltage)** and **CC (Constant Current)** modes, simulating real sensor feedback in C++. Also mapped out a full hardware build (Arduino Nano, INA219 sensor, DPS5005 buck converter, LCD) plus a GUI for real-time monitoring.

`C++` `Feedback Control` `Arduino` `Power Electronics`

</td>
<td width="33%" valign="top">

### ⚡ Maxwell Inductance Bridge

Designed, derived, and physically built a **Maxwell Bridge circuit** to measure an unknown inductance against a known standard. Verified via hand-derived balance equations, Proteus simulation, and a real breadboard build — measured value landed within close range of the expected 330 µH.

`Circuit Analysis` `AC Bridges` `Proteus` `Hardware`

</td>
<td width="33%" valign="top">

### 💡 LED Intensity Control

Designed and built a circuit to control **LED brightness** by varying current/voltage delivery — laying the groundwork for PWM-based control, a core concept in motor drivers, EV battery systems, and embedded lighting.

`Circuit Design` `Analog Electronics` `PWM Fundamentals`
